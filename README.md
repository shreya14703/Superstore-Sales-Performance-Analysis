# 📊 Superstore Sales Performance Dashboard – Excel Project

## 📌 Project Overview
This project showcases an interactive **Sales Performance Dashboard** built in **Microsoft Excel** using the Superstore dataset. The dashboard enables comprehensive analysis of sales, customer segments, product categories, regions, and shipping methods.

The primary goal of this project is to transform raw transactional data into actionable business insights using **Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting**.

---

## 🎯 Objectives
- Analyze total sales, orders, and trends across regions and categories.  
- Understand customer behavior by segment, shipping preferences, and product subcategories.  
- Visualize sales performance by **Region, Category, Subcategory, Segment, and Product**.  
- Enable interactive filtering using **Slicers** for dynamic insights.  
- Identify key areas for business improvement and optimization.

---

## 📁 Dataset Description
The dashboard is built on a **Superstore sales dataset** containing transactional-level details:

| Column Name       | Description                                      |
|------------------|--------------------------------------------------|
| Row ID            | Unique identifier for each row                  |
| Order ID          | Unique identifier for each order                |
| Order Date        | Date of the order                                |
| Ship Date         | Date when order was shipped                      |
| Ship Mode         | Shipping mode (First Class, Standard, etc.)     |
| Customer ID       | Unique customer identifier                       |
| Customer Name     | Customer full name                               |
| Segment           | Customer segment (Consumer, Corporate, Home Office) |
| Country           | Country of the customer                           |
| City              | City of the customer                             |
| State             | State of the customer                            |
| Postal Code       | Customer postal code                             |
| Region            | Region of the order (East, West, South, Central)|
| Category          | Product category (Furniture, Office Supplies, Technology) |
| Sub-Category      | Product subcategory                               |
| Product ID        | Unique product identifier                         |
| Product Name      | Name of the product                               |
| Sales             | Revenue generated by the order                    |
| Quantity          | Number of units sold                              |
| Discount          | Discount applied to the order                     |
| Profit            | Profit earned from the order                      |

## 📊 Dashboard Features

### 1. KPI Metrics
- **Total Sales** – Displays the total revenue generated.  
- **Monthly Sales Trends** – Line chart showing how sales fluctuate across months.  

### 2. Sales & Profit Analysis
- **Sales by Region** – Visualizes sales across the East, West, South, and Central regions.  
- **Sales by Category & Subcategory** – Compares revenue for Furniture, Office Supplies, Technology, and detailed subcategories.  
- **Sales by Customer Segment** – Shows performance by Consumer, Corporate, and Home Office segments.

### 3. Advanced Filtering (Slicers)
Users can dynamically filter the data by:  
- Order Date & Ship Date  
- Region & State  
- Category & Subcategory  
- Customer Segment & Name  
- Ship Mode  
- Product ID & Name  

This enables interactive and granular exploration of the dataset.

### 4. Insights Section
The dashboard includes a pre-built **insights panel**, highlighting key observations such as:  
- West region has the highest orders.  
- Technology category generates the highest sales.  
- Consumer segment contributes the maximum orders.  
- Standard Class is the most preferred shipping mode.  
- East region has the highest number of orders, while Central region has the lowest.

## 🔍 Key Insights
- **Region:** West and East are top-performing regions for sales.  
- **Category:** Technology leads in sales, followed by Furniture and Office Supplies.  
- **Customer Segment:** Consumer segment contributes the most orders.  
- **Shipping Mode:** Standard Class is the preferred choice for most orders.  
- **Trend Analysis:** Sales peak during specific months, which can guide inventory planning.

## 🚀 How to Use
1. Open the Excel file in **Microsoft Excel 2016 or later**.  
2. Navigate to the **Dashboard** sheet.  
3. Use the **Slicers** to filter data by Region, Category, Segment, Product, and Date.  
4. Observe dynamic updates in charts, KPIs, and insights panel.  
5. Analyze trends, top-performing regions/categories, and segment-specific performance.
